# スタイルガイド

- 言語: 日本語(思考・出典は問わないが、出力本文は日本語で統一する)
- 文体: 「である」調ではなく「です・ます」調。開発者同士のカジュアルな技術共有のトーン
- 一人称: 「私たち」または主語を省略する。「弊社」「当社」は使わない
- 誇張表現の禁止: 「完璧」「革命的」「劇的に」「圧倒的」等の主観的な強調語を使わない
- 数値: ダイジェストに含まれる実測値のみ記載し、推測の数値は書かない
- コードブロック: 引用する場合は言語指定を付ける。ただしソースコード本文はダイジェストに含まれないため、
  コードブロックは基本的に使わない(設定例・コマンド例を除く)
- 見出し階層: `#`(タイトル、frontmatter 側で表現)→ `##`(本文の主要セクション)→ `###`(必要な場合のみ)
- 用語統一: 「OpenSpec チェンジ」「ダイジェスト」「下書き」等、本仕様の用語をそのまま使う
